编写一个程序判断给定的数是否为丑数。
丑数就是只包含质因数 2, 3, 5 的正整数。
示例 1:
输入: 6
输出: true
解释: 6 = 2 × 3
示例 2:
输入: 8
输出: true
解释: 8 = 2 × 2 × 2
示例 3:
输入: 14
输出: false 
解释: 14 不是丑数，因为它包含了另外一个质因数 7。
说明：
1 是丑数。
输入不会超过 32 位有符号整数的范围: [−231,  231 − 1]。

## 思考
  丑数就是可以被2,3,5除尽的数(是三者，并不是单独的一个数,并且只能有这三者)
  例如15 true 可以被 3 5 除尽
      14 false 14/2 = 7  7 不行 
      16 true 16/2 = 8
      29 false
  先判断输入的数对2 3 5取余是否等于0，等于的话就除以当前的数
